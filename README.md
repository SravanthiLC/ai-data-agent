AI Data Agent

Download dataset : https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce
Unzip the dataset and transfer the 9 CSV files into `ai-data-agent/data` folder.

Create a python virtual environemnt.
```bash
sudo apt update
sudo apt install python3-venv
python -m venv v1
```

Activate the virtual environment.
```bash
source v1/bin/activate
```

Once the virtual environment is activated, install required dependencies.
```bash
pip install -r requirements.txt
```
